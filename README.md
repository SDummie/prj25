#Beyond Booking 2.0 – Hotel Management System
“A full-stack web application aimed at streamlining hotel operations through efficient booking management, optimized room search, and scalable backend services.”
Tech Stack: React, Node.js, Express.js, PostgreSQL, JavaScript, BST, AVL Trees
• Designed and developed a complete hotel management platform where users can search rooms, make bookings, manage guest records, and track maintenance activities.
• Engineered a RESTful backend using Express.js with modular routes for bookings, rooms, guests, admin, and maintenance management.
• Implemented Binary Search Tree and AVL Tree data structures to optimize room indexing and improve search and retrieval performance.
• Developed a multi-dimensional search system to filter rooms based on price, capacity, and other attributes.
• Designed relational database schemas in PostgreSQL with indexing to enhance query efficiency and data consistency.
• Focused on clean project structure and scalable API architecture, enabling easy integration of new features and system expansion.
